# Cat Prediction


## Setup


Ensure Python 3, `virtualenv` and `pip` are installed.

```console
which python3

virtualenv -p `which python3` venv
source venv/bin/activate
python --version
pip --version
pip install -r requirements.txt 
```

## Running JupyterLab

```console
jupyter lab
```

## Streamlit


```console
streamlit run cat_predictor_app_simple.py
```
